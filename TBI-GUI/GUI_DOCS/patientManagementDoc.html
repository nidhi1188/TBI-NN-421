Patient Management

The purpose of this static class is to manage patient storage and encryption and provide an easy mechanism for saving and refreshing data.

Creating a storage system for patient data required us to consider HIPAA regulations for storage of identifiable patient records.

We addressed each of the HIPAA required technical safeguards in the following ways:
- Access Control: This program will be on PCH computers that require a valid login in order to use. Therefore, there is no reason to password-protect the program.
- Audit Controls: This program will be covered by the access logging systems already implemented on PCH systems.
- Integrity Controls: Patient health information used by this system is stored in an encrypted and serialized format. We have selected AES as our encryption standard, as it is one of the strongest currently available and is used worldwide. Users will not be able to access, modify, or destroy the data through improper channels.
- Transmission Security: This program does not transmit information over any network.


String buildDefaultPath()
Creates the default path for the patient storage directory

Inputs:

- None

Returns:

- The full path to the program directory src/resources/patients

 String getDefaultPath()
Returns the current default path for the patient storage directory

Inputs:

- None

Returns:

- The current default path

 boolean exportPatient(Patient patient)
Encrypts and saves a patient to the default path. Creates a new entry in storage if the patient has not been saved before or overwrites previous data if the patient already exists.

Inputs:

- patient: The Patient object to be saved

Returns:

- A boolean indicating whether the export was successful

Throws:

- IOException if the write fails

 Patient importPatient(String path, String uid)
Decrypts and loads a patient from the provided path.

Inputs:

- path: The path that the patient's directory is stored at

- uid: The patient's UID

Returns:

- The patient loaded from memory

Throws:

- IOException if the read fails

 void addPatient(PatientEntry p)
Adds a new patient entry to storage

Inputs:

- p: The patient entry that will be added

Returns:

- None

Throws:

- IOException if the write fails

 void exportPatientList()
Writes the current list of patient entries to storage

Inputs:

- None

Returns:

- None

Throws:

- IOException if the write fails

 Hashtable importPatientList()
Updates and returns the currently stored list of patients

Inputs:

- None

Returns:

- Hashtable with list of all patients stored (key=uid,value=PatientEntry)

Throws:

- IOException if the read fails

 Hashtable getPatientList()
Gets the list of patients currently loaded into memory

Inputs:

- None

Returns:

- Hashtable with list of all patients stored (key=uid,value=PatientEntry)
