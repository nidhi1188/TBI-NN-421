InfoEntry

The purpose of this static class is to provide easy encryption methods for multiple forms of information. Code can be found on Github.


Methods:
int exportInfo(Patient patient, Key key)
Encrypt a patient and export it to the proper file. Returns the associated key for metadata storage.

Inputs:

- Patient patient

- Key key

Returns:

- Key
