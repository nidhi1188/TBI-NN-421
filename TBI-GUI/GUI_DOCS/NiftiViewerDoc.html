Nifti Viewer GUI Page

The purpose of this is to create a window that will open, allowing
the user to view Nifti images slice by slice. This is accomplished using
the nibable and matplotlib python modules to render and display
different elements.

You will need Python 3.6 or later installed and added as a PATH variable
in order to run this viewer.
You will also need to install the following modules to run the niftiviewer
Module Requirements:
  • nibabel
  • matplotlib
Once these are installed, you will be able to open the niftiviewer
through the normal flow of the program.
