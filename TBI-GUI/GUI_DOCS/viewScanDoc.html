View Scan GUI Page

The purpose of this is to layout the view scan page and fill it with
elements pertaining to the view scan page. This is accomplished using
the Javafx library for all GUI elements.

Scene initializeScene(StateManager manager)
Constructor for View Scan Scene

Inputs:

- StateManager: object to manage state of the UI

Returns:

- A scene filled with elements pertaining to the view scan page

void styleButton(Button button)
Applies a style to the buttons on the page.

Inputs:

- button: the button to be styled in the UI
