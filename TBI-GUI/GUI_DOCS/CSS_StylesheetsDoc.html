CSS Stylesheet Page

The purpose of this is apply reoccurring styles to different
elements of the application. This can be seen in our buttons
as well as the different backgrounds throughout the program.
This coupled with our builder design pattern allows creation
of repeating elements easily.

Stylesheets being more widely accepted as industry practice over
inline CSS, our project will employ the same practice.
Just the same as we have used the style class in the past, we will
be able to accomplish styling using one line:
- obj.getStyleClass().add("style_defined_in_css_file");

This will also allow for easily switching between different themes
should we ever decide to make them
