Likely Trauma Area GUI Page

The purpose of this is to layout the Likely Trauma Area page and fill it with
elements pertaining to the it. This is accomplished using
the Javafx library for all GUI elements and the builder design pattern
to create similar or repeating elements.

Scene initializeScene(StateManager manager)
Method for Likely Trauma Area Scene construction

Inputs:

- StateManager: object to manage state of the UI

Returns:

- A scene filled with elements pertaining to the Likely Trauma Area page
