State Manager Page

The purpose of this is to have a centralized object to control
setting the stage for the UI. This also includes being a
container for variables used throughout the UI and being
responsible for painting the scenes to the stage.

void paintScene(String sceneID)
Method for setting the stage to a different scene.

Inputs:

- SceneID: unique string for one of the scenes

Returns:

- Nothing, but changes the current scene being displayed
