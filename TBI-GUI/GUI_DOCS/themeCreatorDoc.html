Theme Creator GUI Page

The purpose of this is to allow users to create their own themes for
the app. This is accomplished by using the Javafx library for all GUI
elements. Use the color pickers to pick colors for the different UI
elements.

Scene initializeScene(StateManager manager)

Method for Theme Creator Scene construction
Inputs:
- StateManager: object to manage state of the UI
Returns:
- A scene filled with elements pertaining to the theme creator page

String toHex(Color color)

Method for converting a Color object from a ColorPicker into a hex String
Inputs:
- Color: object holding a color
Returns:
- A String representing the color in hex

void writeCss(StateManager manager, String name, HashMap colors)

Inputs:
- StateManager: object to manage state of the UI
- String: a name chosen for the new theme
- HashMap: holds the colors as values mapped to keys representing the elements
Returns:
- none
