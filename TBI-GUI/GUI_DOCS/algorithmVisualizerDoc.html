Algorithm Visualizer GUI Page

The purpose of this is to layout the algorithm visualizer page and fill it with
elements pertaining to the algorithm visualizer page. This is accomplished using
the Javafx library for all GUI elements. The algorithm visualizer is intended to
show graph(s) detailing the accuracy of the machine learning algorithm.

class Point
inner class featuring x and y values
Variables:
  • num: the number of files analyzed (x value)
  • percent: percent accuracy of the algorithm (y value)
Scene initializeScene(StateManager manager)
Constructor for View Scan Scene
Inputs:
  • StateManager: object to manage state of the UI
Returns:
  • A scene filled with elements pertaining to the algorithm visualizer page
LinkedList<Point> importData()
Imports computation data from the machine learning algorithm
Inputs:
  • none
Returns:
  • A linked list of points to be used in the graph(s)
LineChart<Number, Number> getChart(boolean displayRecent)
Imports computation data from the machine learning algorithm
Inputs:
  • displayRecent: boolean value indicating how many of the points to render on the graph
Returns:
  • A line chart with the proper points added to it
