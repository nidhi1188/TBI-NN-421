BSSCS DICOM-Nifti Conversion Tools

class prepare_scans

For public access. Pulls together other methods and handles common errors.

decompress_and_convert(compressed_path)
Convert a compressed DICOM file to Nifti format

Input:

- compressed_path: file path to the folder containing compressed DICOM images

Returns:

- path to new Nifti file
convert(input_path)
Converts a raw DICOM file to Nifti format

Input:

- compressed_path: file path to the folder containing raw DICOM images

Returns:

- path to new Nifti file
decompress(input_path)
Decompresses a compressed DICOM file

Input:

- path: file path to the folder containing compressed DICOM images

Returns:

- path to folder containing raw DICOM images

class DICOM_2_NIFTI

Not to be used publicly unless necessary. Converts raw DICOM files to Nifti format.

convert_dicom_series(input_path)
Converts a raw DICOM file series into a Nifti file. Saves a Nifti file to the file system based on path of DICOM file.

Input:

- input_path: file path to the folder containing raw DICOM images

Returns:

- path to new Nifti file
get_nifti_array(nifti_path)
Converts a Nifti file into a numpy array

Input:

- nifti_path: file path to Nifti image

Returns:

- Numpy array representative of Nifti image

class decompress_DICOM

Not to be used publicly unless necessary. Decompresses compressed DICOM files.

decompress_series(input_path)
Decompresses compressed DICOM files and stores them based on path of compressed files

Input:

- input_path: file path to the folder containing compressed DICOM images

Returns:

- path to folder containing raw DICOM images
decompress_single(input_path)
Decompresses compressed DICOM files and stores them based on path of compressed files

Input:

- input_path: file path to a valid, compressed DICOM images

Returns:

- path to raw DICOM image
