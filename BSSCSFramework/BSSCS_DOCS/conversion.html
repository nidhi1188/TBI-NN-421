BSSCS DICOM-Nifti Conversion Tools

Converting DICOM images to NiFti format was a roadblock, as none of our team members have experience with medical imaging technology. We overcame this through extensive research into both DICOM and NiFti formats.
A major part of this research went into selecting an accepted conversion technology that would retain detail and information without overcomplicating the process.
We chose to use an external package due to our inexperience with the content area, but made sure to gain an understanding of the inner workings of conversion technology so that we could make an educated decision.
We considered packages listed here, based on resources provided by the sponsor. The packages we saw as the most likely candidates were dcm2niix, dicom2nifti, and dcmstack.
We eventually landed on dicom2nifti as the best choice, as it is written in Python, uses packages already necessary for other functions in the framework, and has more extensive documentation than the other choices.
This requires us to manually decompress any compressed images before they are converted, but we accomplished that through an additional Python script.

class prepare_scans

For public access. Pulls together other methods and handles common errors.

decompress_and_convert(compressed_path)
Convert a compressed DICOM file to Nifti format

Input:

- compressed_path: file path to the folder containing compressed DICOM images

Returns:

- path to new Nifti file
convert(input_path)
Converts a raw DICOM file to Nifti format

Input:

- compressed_path: file path to the folder containing raw DICOM images

Returns:

- path to new Nifti file
decompress(input_path)
Decompresses a compressed DICOM file

Input:

- path: file path to the folder containing compressed DICOM images

Returns:

- path to folder containing raw DICOM images

class DICOM_2_NIFTI

Not to be used publicly unless necessary. Converts raw DICOM files to Nifti format.

convert_dicom_series(input_path)
Converts a raw DICOM file series into a Nifti file. Saves a Nifti file to the file system based on path of DICOM file.

Input:

- input_path: file path to the folder containing raw DICOM images

Returns:

- path to new Nifti file
get_nifti_array(nifti_path)
Converts a Nifti file into a numpy array

Input:

- nifti_path: file path to Nifti image

Returns:

- Numpy array representative of Nifti image
